# NL2SQL-Chatbot-using-Vanna-AI-FastAPI
This project is a Natural Language to SQL (NL2SQL) Chatbot that allows users to interact with a database using plain English queries. Instead of writing complex SQL queries, users can simply ask questions, and the system automatically generates SQL, executes it, and returns the results.


Key Features
✅ Convert natural language into SQL queries
✅ Execute SQL queries on a database automatically
✅ FastAPI-based REST API
✅ Uses AI (Vanna AI + OpenAI) for intelligent query generation
✅ SQLite database integration
✅ Easy-to-use Swagger UI for testing

Tech Stack
Backend: FastAPI
AI Engine: Vanna AI + OpenAI API
Database: SQLite
Vector Store: ChromaDB
Language: Python
